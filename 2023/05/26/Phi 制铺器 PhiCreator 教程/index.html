这次 我们将使用简单易用的 PhiCreator 编辑器来制作你的 Phigros 自制铺
不过 使用 PhiCreator 制作的铺面并不能游玩 只能用来演示 如果想体验完整的乐趣 请使用 PhiEditer

1. 下载 PhiCreator

PhiCreator 是 Github 上的一个开源项目，由 AsakuraMizu 构建，支持 Linux & Win 系统

可惜的是 该项目目前已停更 最后一次更新不得不停留在 22 年 6 月

2. 创建一个铺面

准备好如下文件
  • 铺面音乐（ 最好是 .wav 或 .ogg 格式 命名为 “music” ）
  • 铺面背景（ 横板 16 : 9 1080p 或以上分辨路 命名为 “background” ）

新建一个目录，将准备好的文件移动到里面
打开 PhiCreator，进入 “ META ” 选项卡，点击 “ CREATE/OPEN PROJECT “ ，选择新建的目录，打开它，依次填入：

Title :  铺面标题
Difficulty :  铺面难度（ 例 : IN Lv.12 ）
Artist :  作曲家
Illustrator :  插画家
Chrter :  铺师
Music :  ( 音乐文件在目录内的相对路径 一般以 ./ 开头 )
Background :  ( 背景文件在目录内的相对路径 一般以 ./ 开头 ) 

点击 “SAVE” 关闭 PhiCreator ，重新打开并加载铺面
